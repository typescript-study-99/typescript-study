## 2. 규칙
***
### 2-1. 기출 문제 만들기
1. 수요일 오후 9시 미팅 전까지 최소 3문제 이상 깃허브에 올려주세요!
2. 문제의 주제는 각주차의 강의 주제와 동일하게 해주세요

### 2-2. 개인 폴더에 업로드하기
1. branch는 별도 관리 없이 main에서 작업하셔도 됩니다
2. 다만 충돌을 방지하기 위해 꼭 개인 폴더에서만 작업해주세요!
3. 파일명 / 파일구조는 정해진 방식이 없습니다. 편하신대로 작성해주세요
